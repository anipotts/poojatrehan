# Sound Effects

## success-ding.mp3

This file contains the success chime that plays when magic word login is successful.

### Requirements:
- **Format**: MP3
- **Duration**: 200-400ms
- **Volume**: Normalized to -6dB
- **Size**: < 50KB
- **Style**: Pleasant bell chime or notification sound

### Where to Source:

**Option 1: Freesound.org (Recommended)**
1. Visit https://freesound.org
2. Search for: "notification bell", "success chime", or "gentle ding"
3. Filter by:
   - License: Creative Commons 0 (CC0) for unrestricted use
   - Duration: 0-1 seconds
   - Format: MP3 or WAV
4. Download and place in this directory as `success-ding.mp3`

**Option 2: Free Sound Libraries**
- Zapsplat: https://www.zapsplat.com (free with attribution)
- Mixkit: https://mixkit.co/free-sound-effects/ (free, no attribution)
- Pixabay: https://pixabay.com/sound-effects/ (free, no attribution)

**Option 3: Generate Your Own**
- Use online tone generators to create a simple bell sound
- Export as MP3 with the specifications above

### Recommended Sounds:
- "Notification Bell Chime" - Clean, professional
- "Success Ding" - Positive, upbeat
- "Gentle Bell" - Soft, pleasant

### Note:
The implementation respects `prefers-reduced-motion` and will not play sound if the user has motion reduction enabled.
